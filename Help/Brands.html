Getting Started ››

You can add/edit/delete models and related models on this page.  Always add a brand or a model name in English and then add its language translation on language tags page.

For ex, If you added a new brand or model then you'll have to add the tag and its translation on language tag page otherwise the brand and model won't appear in the search form or add listing form.


So if you are added a new brand or model "my category" and your default site language is French then add it on language tag page like this:

Keyword:  my category

Translation: ma catégorie


To summarize, Make sure that you add new brand or model name in English only on 'Brands' page and then add its translation on 'Language tags' page.
