Features

Getting Started ››

On this page the site administrator can add or delete features that will be visible in the add and edit listing form. For ex, if you want certain features to appear on the 'add / edit listing' form then you can add them here. Similarly you can removes the ones that you don't want to see in the add / edit listing form.


Please make sure that you add the new features name in English and then if your default site language is not English then add its translation on 'Language tag' page also. You won't see the feature, if you don't preform this step.
