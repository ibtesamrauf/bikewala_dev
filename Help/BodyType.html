Body Type

Getting Started ››

Site admin can add or delete the auto body types that will appear in the search form, add / edit listing form. Please make sure that you add the new body type in English and then if your default site language is not English then add its translation on 'Language tag' page also. You won't see the body type, if you don't preform this step.
